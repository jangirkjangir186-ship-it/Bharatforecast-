#  Weather Forecast System

##  Description

Weather Forecast System ek simple aur useful application hai jo users ko kisi bhi city ka real-time weather check karne me help karta hai. Iska main goal fast aur accurate weather information provide karna hai, jisse user apne daily plans easily manage kar sake.

##  Features

* City search functionality
* Real-time temperature display
* Humidity information
* Weather condition (Sunny, Rainy, Cloudy, etc.)
* Live data using weather API

##  Technologies Used

* HTML
* CSS
* JavaScript

##  How It Works

User city ka naam enter karta hai, uske baad application weather API se data fetch karta hai. Yeh data user ko temperature, humidity aur weather condition ke form me show hota hai. Isse user ko instant aur accurate weather information mil jati hai.
